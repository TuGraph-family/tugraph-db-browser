TuGraph DB BROWSER 🔗
TuGraph DB BROWSER 是 TuGraph 图数据库的可视化工具。可以完成图谱、模型、数据等的创建和导入。同时可用使用 TuGraph Cypher 进行数据的操作。

0. 环境准备

- node.js
- python2

1. 安装项目

```bash
yarn run bootstrap
```

2. 启动客户端

```js
cd client
yarn start // 切记，一定要 yarn start，npm run start 会存在若干问题！
```

2. 启动服务端

```bash
cd server
npm run  dec
```

浏览器访问 http://localhost:8888
